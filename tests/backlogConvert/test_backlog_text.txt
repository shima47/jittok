* マークダウンからBacklog記法への変換テスト

** 見出し
'''*''' 見出しレベル3
'''*'''* 見出しレベル4
''*'' 見出しレベル5

** テキスト装飾
''太字テキスト''と'''斜体テキスト'''の例です。
%%打ち消し線%%も確認しましょう。

** リスト
- リスト項目1
- リスト項目2
-- ネストされたリスト項目2-1
--- さらにネストされた項目2-1-1
- リスト項目3

** 番号付きリスト
+ 最初の項目
+ 2番目の項目
+ 3番目の項目

** チェックリスト
- [ ] 未完了のタスク
- [x] 完了したタスク
- [ ] まだやってないタスク

** コードの例
インラインコード: {{{const x = 10;}}}

{code:java}
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}

{/code}

** 複合的な例
